# Ad Package Import Guide

Ad packages can use the existing product import pipeline with ad product types.

Recommended CSV fields:

- `title`
- `slug`
- `product_type`
- `ad_placement_type`
- `placement_key`
- `duration_days`
- `price`
- `recurring_price`
- `recurring_interval`
- `included_impressions`
- `included_clicks`
- `requires_approval`
- `requires_creative_upload`
- `allowed_media_types`
- `dimensions`
- `affiliate_eligible`
- `status`

Advanced ad-specific column mapping can be added in a later import pass.
