# Affiliate Matrix Plan

Date: 2026-05-10

## Purpose

The Matrix structure lets the owner configure a fixed width and depth ambassador network, such as 3 x 7.

## Owner Editable Settings

- Matrix width
- Matrix depth
- Spillover mode
- Completion bonus toggle
- Completion bonus amount
- Level commission mode
- Level labels
- Level percentages
- Per-order and monthly caps

## MVP Status

Matrix is configurable, persisted, previewable, and calculator-ready. The real payout engine remains scaffolded until position placement, spillover behavior, and matrix ancestor calculations are hardened with tests.

## Compliance Guardrail

Matrix commissions must be based only on qualified purchase activity. Joining the program does not create commissionable income by itself.
