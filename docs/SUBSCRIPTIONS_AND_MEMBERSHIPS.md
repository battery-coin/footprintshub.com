# Subscriptions And Memberships

Subscription and membership products use recurring checkout and subscription lifecycle records.

## Implemented

- Product editor recurring settings.
- Checkout mode resolver.
- Stripe Checkout `subscription` mode.
- `SubscriptionPlan`, `CustomerSubscription`, and `SubscriptionEntitlement` schema.
- Webhook handling for subscription update/delete status changes.

## Setup required

Stripe Customer Portal and full subscription self-service management remain setup-required.
