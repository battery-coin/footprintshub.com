# Ad Security Checklist

- Admin approval required by default.
- Target URLs must be valid http/https URLs.
- `javascript:` and unsafe URL schemes are rejected.
- HTML ads are disabled by default unless reviewed and sanitized.
- Creative file uploads should use MIME/type/size validation.
- Policy acceptance should be stored before publishing.
- Click fraud and impression fraud controls are planned.
- No ad goes live before payment verification and admin approval.
